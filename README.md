## Example Telegram bot wich show top crypto currencies

### System requirements 

#### Development

`node v^12.19.0`  

`yarn v^1.22.10`

#### Production
`screen v^4.01`

### Node requirements

Typescript  

`npm i -g typescript`

### Instalation

`git clone https:github.com/kolserdav/crypto-bot`  

`cd crypto-bot`  

`yarn install`  


set up file .env.example and rename it to .env

### Start on develop

`yarn ts`  

`yarn dev`  


### Start on production

`yarn deploy`
